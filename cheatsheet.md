# Git CheatSheet

### STart a new project

```shell
$ mkdir project_name
$ git init
$ touch readme.md
$ git add readme.md
$git commit -m "initial commit"
```

### Do some work and then save it

First, do some work!

```shell
$ git status
$ git add <whatever file>
$ git status
$ git commit -m "I made a bunch of changes, there are so many details, wee."
```
